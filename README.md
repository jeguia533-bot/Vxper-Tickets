# VXPER Applications Bot

Railway-ready Discord ticket bot for VXPER applications.

## Files

- `bot.py`
- `requirements.txt`
- `README.md`

## Railway Variable

Only add this variable in Railway:

```env
DISCORD_TOKEN=your_bot_token_here
```

## How to use

1. Invite the bot to your Discord server.
2. Make sure the bot role is above the staff role.
3. Give the bot permissions to:
   - View Channels
   - Send Messages
   - Manage Channels
   - Read Message History
4. Run this command in your application channel:

```txt
!panel
```
